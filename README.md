## 🎬 مولّد أوامر FFmpeg

أداة ويب بسيطة تساعدك على توليد أوامر FFmpeg لترميز الصوت والفيديو بإعدادات احترافية، دون الحاجة لحفظ أي أوامر أو معرفة مسبقة بخيارات FFmpeg.

### 🔗 تجربة الأداة
* [🚀 معاينة حية للأداة](https://iofahmawi.github.io/ffmpeg/)

### ✨ المميزات

* **ترميز صوتي متكامل**: دعم ترميزات MP3، Opus، AAC-LC، HE-AAC v1/v2، Vorbis، وFLAC.
* **ترميز فيديو احترافي**: دعم H.264، H.265، وAV1 مع التحكم الكامل في الإعدادات.
* **جودة ثابتة أو معدل بت ثابت**: اختيار بين وضع CBR أو VBR/CRF حسب الحاجة.
* **إعدادات متقدمة**: التحكم في معدل العينة (Sample Rate)، عدد القنوات، ملف التعريف (Profile)، وعدد الإطارات (FPS).
* **وضع الكود الكامل**: خيار لإظهار الأمر كاملاً مع مسار الإدخال والإخراج لبعض التطبيقات.
* **نسخ سريع**: نسخ الأمر الناتج بضغطة واحدة.
* **تصدير JSON**: تصدير الإعدادات كملف JSON جاهز للاستخدام في تطبيقات أخرى.
* **واجهة عربية كاملة**: تصميم نظيف ومتجاوب مع جميع أحجام الشاشات.

### 🛠 التقنيات المستخدمة

* **HTML5 & CSS3 & JavaScript (Vanilla)**: بدون أي أطر عمل خارجية.

### 📊 الترميزات المدعومة

| النوع | الترميزات |
|------|-----------|
| صوت | MP3 (libmp3lame)، Opus، AAC-LC، HE-AAC v1/v2، Vorbis، FLAC |
| فيديو | H.264 (libx264)، H.265 (libx265)، AV1 (libaom-av1) |

### 🚀 كيفية الاستخدام

1. **اختر نوع الترميز**: تبويب "ترميز صوتي" أو "ترميز فيديو".
2. **حدد الإعدادات**: اختر الترميز، نمط الجودة، والخيارات المناسبة.
3. **الكود الكامل** *(اختياري)*: فعّل الخيار لإضافة مسار الإدخال والإخراج للأمر.
4. **انسخ أو صدّر**: انسخ الأمر مباشرة أو صدّره كملف JSON.

### 📂 ملفات المشروع

* `index.html`: الملف الأساسي الذي يحتوي على الواجهة والمنطق البرمجي.
* `manifest.json`: ملف تعريف التطبيق للتثبيت على الأجهزة.
* `sw.js`: ملف Service Worker للتشغيل أوفلاين.
